# Contributing

Pr0jectZer0 is currently a proprietary product under active private development.

This public repository is documentation-only and does not accept source-code contributions.

Helpful public contributions may include:

- Reporting documentation errors
- Suggesting clearer public explanations
- Reporting broken public links
- Submitting responsible security reports through the process in `SECURITY.md`

Do not submit proprietary code, reverse-engineered implementation details, customer data, credentials, vulnerability data from customer environments, or private development materials.

Opening an issue does not create a support obligation or guarantee that a request will be accepted.
